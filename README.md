# OpenMC-Installation-Guide-Conda-Jupyter-

This repository provides a step-by-step guide to installing OpenMC from scratch using Conda. It is designed for students, researchers, and engineers who want a reliable, reproducible setup for nuclear simulation workflows.

The guide is delivered as an interactive Jupyter Notebook (.ipynb), making it easy to follow, copy commands, and verify each step.

🚀 What This Guide Covers
- Installing Miniconda / Anaconda
- Creating an isolated Conda environment
- Configuring conda-forge
- Installing OpenMC
- Downloading and setting up nuclear data libraries
- Configuring environment variables
- Verifying the installation
- Optional Jupyter kernel setup
- Troubleshooting common issues

💻 Supported Platforms
✅ Linux (recommended)
✅ macOS
⚠️ Windows (via WSL2 only)

OpenMC is not natively supported on Windows — WSL2 is strongly recommended.

⚙️ Requirements
Conda (Miniconda or Anaconda)
≥ 4 GB RAM (8 GB recommended)
≥ 5 GB free disk space
Internet connection

🧠 Who Is This For?
* Nuclear engineering students
* Researchers working with Monte Carlo simulations
* Anyone new to OpenMC who wants a smooth setup
